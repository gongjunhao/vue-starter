# AGENTS.md

## Project Context
This is a frontend starter project. AI coding agents should prioritize consistency and simplicity.

## Core Principles
- Keep UI changes minimal and consistent.
- Do not introduce new frameworks or state systems unnecessarily.
- Preserve existing component structure.

## Code Style
- Follow existing Vue project conventions.
- Use existing components and patterns.
- Avoid over-engineering.

## Development Rules
- Check existing components before creating new ones.
- Prefer composition over duplication.
- Maintain backward compatibility.

## Testing & Quality
- Do not remove existing tests.
- Ensure build passes after changes.
- Validate UI behavior manually if needed.

## Commit Style
- Use simple, descriptive messages.
- Example: fix: adjust header layout
